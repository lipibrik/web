2.4.1. 📖 Guía de Elaboración de Contenidos Web

Guía de Elaboración de Contenidos

Crear y utilizar contenido multimedia propio y libre de derechos de autor es fundamental para el desarrollo web. Esta guía te proporciona los pasos y recursos para generar imágenes, audio y vídeos, asegurando su correcta inserción en sitios web.

Generar Imágenes Propias

  1. Uso de Herramientas de Diseño:
    • GIMP: Software gratuito y de código abierto para la edición de imágenes.
    • Canva: Plataforma online con herramientas de diseño gráfico intuitivas.
  2. Prácticas Recomendadas:
    • Mantén las dimensiones adecuadas y la resolución óptima para web (72 DPI).
    • Optimiza el tamaño del archivo sin comprometer la calidad, utilizando herramientas como TinyPNG.

Insertar Imágenes Libres de Derechos de Autor

  1. Bancos de Imágenes Gratuitos:
    • Unsplash: Amplia selección de fotos de alta calidad libres de derechos.
    • Pixabay: Ofrece tanto imágenes como vectores y vídeos sin restricciones de copyright.
  2. Cómo Insertar:
    • Al seleccionar una imagen, asegúrate de revisar y cumplir con las licencias de uso.
    • Para insertar en tu web, usa la etiqueta HTML <img src="URL_de_la_imagen" alt="descripción">, reemplazando URL_de_la_imagen por el enlace directo de la imagen.

Generar Audio

  1. Grabación de Audio:
    • Usa dispositivos con buena calidad de micrófono o micrófonos externos para grabaciones claras.
    • Audacity: Programa gratuito para grabar y editar audio.
  2. Uso de Audio Libre de Derechos:
    • Free Music Archive: Fuente de música y efectos de sonido libres de derechos.
    • Asegúrate de cumplir con las licencias específicas para uso comercial o no comercial.

Generar Vídeos

  1. Creación de Vídeos:
    • Planifica tu contenido. Escribe un guion o storyboard si es necesario.
    • Utiliza tu smartphone o cámara digital para grabar. Asegúrate de tener buena iluminación y sonido claro.
    • DaVinci Resolve: Herramienta gratuita y profesional para edición de vídeo.
  2. Incorporación de Vídeos Libres de Derechos:
    • Pexels Vídeos: Ofrece vídeos gratuitos que puedes usar en tus proyectos web.
    • Al igual que con las imágenes, verifica las licencias y utiliza la etiqueta HTML adecuada para insertar vídeos: <video src="URL_del_vídeo" controls>Texto alternativo si el vídeo no puede ser mostrado.</video>.
