2.1. 📖 HTML y CSS

Introducción

El HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Es la base sobre la que se construyen todos los sitios web y es esencial para cualquier persona interesada en el desarrollo web. Para profundizar en cada tema de HTML, utilizaremos w3schools como nuestro principal punto de referencia, un recurso educativo en línea reconocido por su claridad y profundidad en temas de programación web.

Creando un documento HTML

  1. Estructura de un Documento HTML: Todo documento HTML comienza con la declaración <!DOCTYPE html>, seguida de la etiqueta <html> que envuelve todo el contenido de la página. Dentro de <html>, encontramos dos secciones principales: <head>, que contiene metadatos y enlaces a hojas de estilo o scripts, y <body>, donde se escribe el contenido visible de la página.

    Referencia: Estructura HTML Básica en w3schools

  2. Creación de Archivos HTML: Para comenzar, creamos un archivo nuevo con la extensión .html. Esto se puede hacer usando cualquier editor de texto (como Notepad) o un Entorno de Desarrollo Integrado (IDE) como Visual Studio Code o Sublime Text.

    Referencia: Cómo crear un archivo HTML en w3schools

  3. Visualizando el HTML: Una vez que hemos escrito algo de HTML en nuestro archivo, podemos ver cómo se muestra en un navegador web. Simplemente abrimos el archivo con cualquier navegador (Chrome, Firefox, Safari, etc.) para ver el resultado.

Creando una página web básica

  • Título y Párrafos: Comenzamos añadiendo un título con la etiqueta <title> en el <head> y párrafos en el <body> usando <p>.

    Referencia: Títulos y párrafos en w3schools

  • Enlaces e Imágenes: Añadimos enlaces con <a href="URL"> y mostramos imágenes con <img src="url_de_la_imagen" alt="descripción">.

    Referencia: Enlaces e Imágenes en w3schools

  • Listas y Tablas: Incorporamos listas utilizando <ul> para listas no ordenadas y <ol> para listas ordenadas. Las tablas se crean con <table>, <tr> para filas y <td> para celdas.

    Referencia: Listas y Tablas en w3schools. Ejemplo:

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico
Ernst Handel Roland Mendel Austria
Island Trading Helen Bennett UK
Laughing Bacchus Winecellars Yoshi Tannamuri Canada
Magazzini Alimentari Riuniti Giovanni Rovelli Italy

  • Cabeceras: Usamos <h1>, <h2>, <h3>, etc., para definir encabezados de diferentes niveles.

    Referencia: Cabeceras en w3schools

Introducción al estilo con CSS

CSS (Cascading Style Sheets) se utiliza para definir el estilo de los elementos HTML. Aprenderemos a cambiar colores, formato de texto y tamaño de imágenes.

  1. Estilos Básicos: Podemos cambiar el color del texto, el fondo de los elementos y el tamaño de las imágenes. Esto se hace utilizando la etiqueta <style> dentro de <head> o en archivos CSS externos.

    Referencia: Etiqueta style en html, css básico en w3schools

  2. Aplicando CSS: Aplicamos estilos directamente a elementos HTML o utilizando clases e IDs para un mayor control.

    Referencia: Aplicación de CSS en w3schools

Ejemplo de aplicación de estilos a un texto:

I am Red

I am Blue
