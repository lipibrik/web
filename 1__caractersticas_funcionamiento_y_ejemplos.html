1. 📖 Características, funcionamiento y ejemplos.

Introducción

Hoy vamos a sumergirnos en el fascinante mundo de la web, un componente crucial de las Tecnologías de la Información y la Comunicación (TIC). La web es el espacio que nos permite acceder a un universo de información y recursos de manera sencilla a través de la red.

La web, también conocida como World Wide Web o WWW, es un sistema de información global en el que se pueden acceder a documentos y otros recursos de medios a través de enlaces de hipertexto. Estos documentos y recursos están interconectados y se acceden mediante la red de Internet.

Conceptos Clave

  1. Documentos y Páginas Web: Los documentos en la web se presentan generalmente como páginas web, que son archivos escritos en lenguajes de marcado como HTML. Estas páginas pueden contener texto, imágenes, videos y otros tipos de contenido multimedia.

  2. Hipertexto: La web utiliza el concepto de hipertexto para enlazar un documento con otro. Esto permite a los usuarios seguir enlaces (links) para navegar de una página web a otra.

  3. Navegadores Web: Para acceder a la web se utilizan programas llamados navegadores web (como Google Chrome, Mozilla Firefox o Safari), que interpretan el código de las páginas web y presentan su contenido de forma legible para los usuarios.

  4. URLs y Dominios: Cada página web tiene una dirección única conocida como URL (Uniform Resource Locator), que generalmente incluye un dominio, como www.ejemplo.com. Estas direcciones permiten encontrar recursos específicos en la vasta red de la web.

  5. Servidores Web: Las páginas y los recursos de la web se almacenan en computadoras especiales conocidas como servidores web. Cuando un usuario quiere ver una página web, su navegador envía una solicitud al servidor web correspondiente para recuperar y mostrar esa página.

  6. Interactividad y Dinamismo: Con el tiempo, la web ha evolucionado para ser más interactiva y dinámica, permitiendo no solo la visualización de información sino también la interacción del usuario, como completar formularios, realizar compras en línea, y participar en redes sociales.

  7. Desarrollo y Tecnologías Web: La creación y el diseño de páginas web involucran varias tecnologías y habilidades, incluyendo HTML para estructura, CSS para diseño y JavaScript para funcionalidades interactivas.

Características de la Web

  • Interconectividad: La web conecta millones de páginas web a través de enlaces, permitiendo navegar de una a otra con facilidad.
  • Hipertexto: Las páginas web utilizan hipertexto, que son textos o imágenes a los que se les puede hacer clic para acceder a otra página o recurso.
  • Accesibilidad: La web es accesible desde cualquier dispositivo con conexión a Internet, como computadoras, teléfonos móviles o tablets.
  • Multimedia: Combina textos, imágenes, videos y sonidos, proporcionando una experiencia rica y dinámica.
  • Interactividad: Permite la participación del usuario, como realizar búsquedas, llenar formularios o participar en redes sociales.

Funcionamiento de la web

  • Servidores Web y Clientes: La web funciona a través de una relación entre el cliente (tu navegador) y el servidor (donde se alojan las páginas web). Cuando escribes una dirección web o haces clic en un enlace, tu navegador solicita esa página al servidor correspondiente.
  • Protocolo HTTP/HTTPS: Es el protocolo de comunicación utilizado para enviar y recibir datos en la web. HTTPS es la versión segura de HTTP.
  • URLs: Son las direcciones de las páginas web, como https://www.ejemplo.com. Cada URL es única y dirige a un recurso específico en la web.
  • HTML, CSS y JavaScript: HTML define la estructura de la página, CSS su estilo y presentación, y JavaScript permite agregar interactividad.

Ejemplos

  • Educación: Plataformas como Coursera o Khan Academy ofrecen cursos en línea.
  • Comunicación: Redes sociales como Facebook o Twitter nos permiten estar conectados.
  • Información: Sitios de noticias como El País o la BBC ofrecen acceso a noticias globales y locales.
  • Comercio Electrónico: Amazon y eBay son ejemplos de cómo se pueden comprar y vender productos en línea.

Conclusión

La web ha revolucionado nuestra forma de comunicarnos, aprender y hacer negocios. Como futuros profesionales, es importante que comprendáis no solo cómo usarla, sino también cómo funciona y se estructura. Ahora, vamos a visualizar esto con una imagen que refleje el mundo interconectado de la web.
